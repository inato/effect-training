# effect-training

## Prerequisites

- install [bun](https://bun.sh)

## Get started

- install dependencies

```sh
bun i
```

- run all solution tests

```sh
bun run test:solution
```

- run all exercise tests

```sh
bun test
```

- run one solution test

```sh
bun run test:solution exo0
```

- run all exercise tests

```sh
bun test exo0
```
